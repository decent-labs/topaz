# Topaz

Topaz is a platform for api-based blockchain interaction

## Getting Started

### Spin up external services

```sh
$ docker-compose up
```

### Run DB migrations if you have to

```sh
$ go run migrations/main.go
```

### Start the API

```sh
$ go run api/main.go
```

### Begin the batch process

```sh
$ go run batch/main.go
```
